# AirPygee
### _The Go RPG Game_

AirPygee is a small golang rpg game using SDL2.


## Features

- Random dungeon sprites generation
- Character move
- Level editor
- Levels interconnection
- Monsters
- Combat

## Build

```
make build
```
## Run

```sh
make run
```
